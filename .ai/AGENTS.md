# Agent Guidelines (AGENTS.md)

This document defines the rules and constraints for AI agents working on this codebase.

## 1. Environment and Tooling
- **JDK Version**: Always use **JDK 21** located at `d:\tools\jdk-21.0.9\`.
- **JDK Modifications**: **NEVER** change the JDK version in `pom.xml` or environment scripts (`mvn.cmd`, etc.) without explicit user approval.
- **Maven**: Use `.\mvn.cmd` with the project-specific `settings.xml`.

## 2. Dependency Management
- **Explicit Approval**: Do not add, remove, or upgrade dependencies in `pom.xml` without clear approval.
- **Classifiers**: When adding Querydsl, always use the `jakarta` classifier to match the Spring Boot 3 / Hibernate 6 stack.

## 3. Code Generation
- **Python Scripts**: The core DAL (entities, repositories, table constants) is generated by Python scripts in `scripts/`.
- **Manual Edits**: Avoid manually editing files in `src/main/java/ch/sunrise/dal/entity/`, `repository/` (except `base`), or `table/` as they will be overwritten.
- **Extension Point**: Use `BaseRepository.java` or `QuerydslPredicateExecutor` for extending repository functionality.

## 4. Architecture
- **Querydsl JPA**: Prefer Querydsl JPA over raw SQL or JDBC.
- **Records**: Use Java Records for read-only projections to avoid Hibernate overhead.
- **Null Safety**: Always use `@NonNull` (Jakarta or Spring) for repository mandatory parameters.
